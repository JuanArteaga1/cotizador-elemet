# Frontend Patterns

Usa esta skill cuando trabajes componentes, interacciones o pantallas del frontend.

## Flujo

1. Lee [patterns.md](patterns.md) para los patrones existentes.
2. Usa clases y convenciones del design system: `wrap`, `card`, `input`, `btn`, `tab`, `small`.
3. Mantén componentes mobile-first, accesibles y coherentes con el diseño oscuro/premium.

## Patrones cubiertos

- Modal de confirmacion.
- Formulario inline.
- Toggle option.
- Service item seleccionable.
- Checkbox item.
- Step indicator y navegacion del wizard.
- Notification toast.
