# Project Context

Usa esta skill al inicio de tareas donde importe el dominio del producto, el stack o el alcance funcional.

## Flujo

1. Lee [project-context.md](project-context.md).
2. Revisa tambien `docs/user-stories.txt` y `docs/design-system.txt` si la tarea afecta UX, flujo de cotizacion o reglas de negocio.
3. Mantén las respuestas y ejemplos en español cuando el trabajo sea de producto.

## Recordatorio

ELEMENT Cotizador es una app profesional para constructoras, arquitectos e ingenieros en Colombia. El tono visual es oscuro, premium, mobile-first y con acento dorado.
