# Project Memory

Usa esta skill cuando necesites saber decisiones ya tomadas, riesgos conocidos o el estado actual del proyecto.

## Flujo

1. Lee [project-memory.md](project-memory.md).
2. No contradigas decisiones existentes sin una razon clara.
3. Si una nueva decision afecta varias tareas futuras, actualiza [project-memory.md](project-memory.md).

## Uso recomendado

Activa esta skill antes de cambios grandes, migraciones, decisiones de stack, persistencia offline o backend.
