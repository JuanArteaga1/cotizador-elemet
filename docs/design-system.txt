================================================================================
  ELEMENT - COTIZADOR PROFESIONAL
  Design System
================================================================================

1. IDENTIDAD DE MARCA
--------------------------------------------------------------------------------
  Nombre:           ELEMENT
  Eslogan:          Cotizador Profesional
  Sector:           Construcción / Arquitectura / Ingeniería
  Tono:             Premium, minimalista, profesional, oscuro, lujoso
  Público:          Constructoras, Arquitectos, Ingenieros civiles en Colombia
  Diferenciador:    Diseño oscuro con acentos dorados, mobile-first, PWA


2. PALETA DE COLORES
--------------------------------------------------------------------------------
  Variable CSS        | Color       | HEX         | Uso
  --------------------|-------------|-------------|-------------------------------
  --bg                | Negro       | #0a0a0a     | Fondo principal de la app
  --card              | Gris oscuro | #1a1a1a     | Fondos de tarjetas y modales
  --text              | Blanco      | #ffffff     | Texto principal
  --muted             | Gris medio  | #999999     | Texto secundario, etiquetas
  --line              | Gris línea  | #2a2a2a     | Bordes, separadores
  --accent            | Dorado      | #b69462     | Acento principal, botones CTA
  --accent-light      | Dorado 15%  | rgba(212,175,55,0.15) | Fondos hover, active
  --danger            | Rojo        | #ff3b30     | Acciones destructivas
  --success           | Verde       | #34c759     | Estados completados, éxito

  Reglas:
  - Botón primario: fondo --accent, texto #000 (negro, no blanco, para contraste)
  - Texto sobre fondos oscuros: siempre --text o --muted
  - Estados "completado/finalizado": usar verde #34c759
  - Errores: usar rojo #ff3b30 con fondos rgba(255,59,48,0.15)


3. TIPOGRAFÍA
--------------------------------------------------------------------------------
  Principal:   -apple-system, BlinkMacSystemFont, 'SF Pro Display', sans-serif
  Estilo:      Sistema nativo Apple San Francisco (SF Pro Display)
  
  Jerarquía:
  - Títulos h1: 32px, weight 700
  - Títulos h3: 18px, weight 600
  - Cuerpo:     16px
  - Small:      12px, color --muted (#999)
  - Botones:    16px, weight 600

  La app delega en la tipografía del sistema operativo (iOS SF Pro,
  Android Roboto/Noto, Windows Segoe UI) para máxima compatibilidad nativa.


4. ESPACIADO
--------------------------------------------------------------------------------
  Unidad base: 4px (sistema Apple HIG)
  
  Padding tarjetas:     20px
  Padding inputs:       14px 16px
  Gap grid:             12px
  Separación inferior:  16px (mb-2), 8px (mb-1)
  Padding wrap:         18px horizontal, 18px top, 120px bottom (para tab)

6. LAYOUT
--------------------------------------------------------------------------------

    - max-width: 430px (formato mobile-first tipo iPhone)
    - margin: auto
    - min-height: 100vh
    - padding: 18px 18px 120px (120px para evitar solapamiento con tab)
  
  Grid:
    - 2 columnas, gap: 12px
  
  Bottom Tab Navigation:
    - Position: fixed, bottom: 20px
    - left: 50%, transform: translateX(-50%)
    - width: calc(100% - 32px), max-width: 400px

    - Padding: 12px

    - Font-size: 14px

7. ANIMACIONES
--------------------------------------------------------------------------------
  - slideDown (forms inline):
    - from: opacity 0, translateY(-10px)
    - to: opacity 1, translateY(0)
    - duration: 0.3s, ease
  
  - notificationSlide:
    - from: opacity 0, translateY(-20px)
    - to: opacity 1, translateY(0)
    - duration: 0.3s, ease
  
  - fadeIn (modal overlay):
    - from: opacity 0
    - to: opacity 1
    - duration: 0.2s, ease
  
  - modalSlide:
    - from: opacity 0, translateY(-20px) scale(0.95)
    - to: opacity 1, translateY(0) scale(1)
    - duration: 0.3s, ease
  
  - Botón active: opacity 0.8, scale(0.98), transition 0.2s
